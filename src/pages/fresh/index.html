Welcome!

Kestrel is a collection of utilities for the browser, combined into a command palette. But of course, since you're already here, you know that already. Let's talk about upgrading your browser experience.

Using Kestrel

Kestrel is activated by holding Alt+Shift+Q. This order was chosen as it is easily able to be activated using a single hand, doesn't conflict with any preexisting shortcuts. Once it's open, it'll look something like the image below (of course, the colour theme may be different).

To close it again, just use the same shortcut as before: Alt+Shift+Q.

Commands can be filtered by typing into the search bar, and can be activated by pressing enter on the currently highlighted command (which is a different colour from the others) or clicking it.

That's all there is to Kestrel. It's a simple, efficient, and functional tool, to help efficient people.

If you want to further customize Kestrel by choosing a theme, enabled commands, or another option, head over to the settings page.

You can also check out what commands do by visiting the reference page.

If you want in on ongoing development or want to check out what's next, view the Kestrel's readme.
